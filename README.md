# niyyah
NIYYAH - Islamic Super App for Malaysian Muslims. Features: Quran (Audio/BM/EN), JAKIM Prayer Times, Authentic Hadith, Doa/Zikir, Iqra/Muqaddam, Muallaf Hub (step-by-step guides + mentor), Online Classes, Community, Zakat Calculator, Islamic Financial Planner. Designed for muallaf &amp; elderly.
